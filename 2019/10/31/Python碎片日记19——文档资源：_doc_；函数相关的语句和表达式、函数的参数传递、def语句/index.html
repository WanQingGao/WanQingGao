函数的参数传递

函数是通过赋值（引用）对象，来使外部变量传入函数中进行计算的。在这里，就会再次出现对象的可变性与不可变性的问题。对于可变对象，在传递后，原变量和传入变量是同一个对象，因此会被直接改变，这点有点像C中的引用传递参数。对于不可变对象，则不会被改变，类似C中的值传递。
