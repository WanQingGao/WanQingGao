迭代协议

这是Python中无处不在的一个概念，表示在内存中物理存储的序列，或一个在迭代操作情况下每次产生一个元素的对象。从左向右扫描一个对象的每个Python工具都使用迭代协议。因此在上面sorted(D)中，sorted调用直接工作于字典值上。
for、while、列表解析等都遵循迭代协议，但列表解析和相关的函数编程工具，如map和filter，通常运行得比for循环快（也许快了两倍）。
